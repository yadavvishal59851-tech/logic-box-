# Logic Box Program (Easy Explanation)

This program is a simple menu-based Python application.\
You can choose what you want the program to do.

------------------------------------------------------------------------

## ‚≠ê Features

### 1. **Generate a Pattern**

-   You enter how many rows you want.
-   The program prints a star pattern like this:

```{=html}
<!-- -->
```
    *
    **
    ***

### 2. **Analyze Numbers in a Range**

-   You enter a starting number and an ending number.
-   The program checks every number in between.
-   It tells whether each number is **Even** or **Odd**.
-   It also adds all odd numbers together and shows the **total sum**.

### 3. **Exit**

-   This option closes the program.

------------------------------------------------------------------------

## üìå How It Works (Simple Steps)

1.  Program shows a menu.
2.  You type a number (1, 2, or 3).
3.  Program performs the selected action.
4.  It repeats until you choose **3 (Exit)**.
